# My website

Made with [Astro](https://astro.build/).

## Toolchain

This project uses [mise](https://mise.jdx.dev/) to install the required local tools:

- Node.js `24.14.0`
- pnpm `10.33.0`

## Quick start

Install the toolchain:

```sh
mise install
```

Start the local dev server:

```sh
mise run dev
```

Preview the production build:

```sh
mise run preview
```

The `mise` tasks will install TypeScript dependencies automatically when needed.
